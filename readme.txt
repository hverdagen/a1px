Application to add slight transparency to the bottom-right pixel of a png.

Zip with installation files: https://www.dropbox.com/s/xvpi02ao10yhmi6/Alpha1px.zip?dl=0
I don't remember why these aren't uploaded on github itself, but it feels almost wrong to modify anything but the readme after this long, so here those are.

How to use: Right-click on a png in the file explorer or from the desktop and from that context menu run it.
